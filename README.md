# Specter UI Kit

## How to clone and set up
1. clone project
2. install dependencies: `npm i`

## How to create new component or icon

```bash
npm run generate
```

and follow instructions


## How to run storybook

```bash
npm run storybook
```

## How to use it
just run
```bash
npm i specter-ui-kit@latest
```

or any version you wanna install

HAVE FUN :)
